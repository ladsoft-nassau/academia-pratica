Git e Github. Para que serve e como usar?

Por: Paulo Lima

O Git é um sistema de versionamento utilizado para manter o histórico de alterações de um arquivo. Podendo assim voltar à determinada versão que se queira.

O GitHub é um portal que armazena repositórios remotos onde desenvolvedores podem observar e interagir com o código de outras pessoas.
